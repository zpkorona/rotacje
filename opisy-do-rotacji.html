1. Liczba wyborów k elementów z n
Lw = n! / (k!*(n-k)!)
2. Liczba wyborów k elementów z n tak by były wszystkie permutacje, czyli każdy element był na każdej pozycji tyle samo razy
Lk = [n! / (k! * (n-k)!)] * k! = n! / (n-k)!
2.a W tym każdy element dla wszystkich kombinacji będzie na 1. miejscu:
Le1 = [(n-1)! / ((k-1)! * (n-k)!)] * (k-1)! = (n-1)!/(n-k)! = Lk / n
2.b czyli jesli każdy element ma być tyle razy na pierwszym miejscu to liczba kombinacji to:
L1 = Le1 * n = [(n-1)! / (n-k)!] * n = n! / (n-k)! = Lk
3. Liczba wystapień każdego z n elementów to:
Le = Lr * k / n, o ile Lr jest wielokrotnością Lw
Lw * k / n = (n-1)! / ((k-1)!(n-k)!)
Lk * k / n = k * (n-1)! / (n-k)!
Permutacja zbioru n - elementów
- obejmuje wszystkie elementy
- kolejność jest istotna
P - liczba permutacji
L - liczba wystąpień każdego elementu jako pierszy
P = n!
L = (n-1)!
Wariacja bez powtórzeń k-elementowa ze zbioru n - elementów
- obejmuje k z n elementów
- kolejność jest istotna
P - liczba wariacji
L - liczba wystąpień każdego elementu jako pierszy
P = n!/(n-k)!
L = (n-1)!/(n-k)!
Kombinacja k-elementowa ze zbioru n - elementów
- obejmuje k z n elementów
- kolejność NIE jest istotna
P - liczba kombinacji
L - liczba wystąpień każdego elementu jako pierszy
P = n!/k!(n-k)!
L = <0;(k-1)>
